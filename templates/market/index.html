TL;DR¶

The distribution of the value of the trades is very concentrated, so is the distribution of the quantity of them.

That is to say, the distribution of the dataset is some what inspiring.

Target¶

Investigate the factors of these Alert Rules:

  1. [AT01] Large-quantity order detected
  2. [AT02] Large-value order detected
  3. [AT24] Large-quantity trade detected
  4. [AT25] Large-value trade detected

Definitions¶

dataset¶

Tick

Order

As a common practise, the Tick dataset is public available, while the Order dataset is not.

(AFAIK, only one exchange, KuCoin, provides the Order dataset by exposing the match engine data through websocket)

For the rules related to trade(AT24, AT25), we can reference from the Tick dataset.

For the rules related to order(AT01, AT02), since we have no Order dataset, we can only infer from Tick dataset

  2. Symbol
